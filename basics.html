1. 什么是 JavaScript?

JavaScript 是一种轻量级的编程语言，主要用于网页开发。它可以让网页具有动态效果和交互功能。

2. 基本语法

2.1. 注释

使用 // 进行单行注释，使用 /* ... */ 进行多行注释。

2.2. 变量

使用 let、const 和 var 声明变量。

let name = "John"; // 使用 let
const age = 30; // 使用 const

2.3. 数据类型

JavaScript 中的基本数据类型包括：

  • 字符串（String）
  • 数字（Number）
  • 布尔值（Boolean）
  • 对象（Object）
  • 数组（Array）

2.4. 控制结构

JavaScript 支持条件语句和循环结构，例如：

if (age > 18) {
    console.log("成年");
} else {
    console.log("未成年");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

2.5. 函数

使用关键字 function 定义函数：

function greet(name) {
    return "Hello, " + name + "!";
}
console.log(greet("John")); // 输出: Hello, John!

2.6. 事件处理

JavaScript 可以处理网页上的事件，例如点击事件：

document.getElementById("myButton").addEventListener("click", function() {
    alert("按钮被点击了！");
});
