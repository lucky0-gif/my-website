1. 什么是事件?

事件是用户与网页交互的行为，例如点击按钮、移动鼠标、键盘输入等。JavaScript 允许我们对这些事件进行响应。

2. 事件类型

常见的事件类型包括：

  • 鼠标事件：click、dblclick、mouseover、mouseout
  • 键盘事件：keydown、keyup、keypress
  • 表单事件：submit、change、focus、blur
  • 窗口事件：load、resize、scroll

3. 添加事件监听器

可以使用 addEventListener 方法为元素添加事件监听器：

const button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("按钮被点击了！");
});

4. 事件对象

事件处理函数的第一个参数是事件对象，包含有关事件的详细信息：

button.addEventListener("click", function(event) {
    console.log(event);
    console.log("点击位置：", event.clientX, event.clientY);
});

5. 事件传播

事件传播有两个阶段：捕获阶段和冒泡阶段。可以通过设置 useCapture 参数控制事件传播：

document.getElementById("parent").addEventListener("click", function() {
    alert("父元素被点击");
}, true); // 捕获阶段

冒泡阶段的设置如下：

button.addEventListener("click", function() {
    alert("按钮被点击");
}); // 默认是冒泡阶段

6. 移除事件监听器

可以使用 removeEventListener 移除事件监听器：

function handleClick() {
    alert("按钮被点击");
}
button.addEventListener("click", handleClick);
button.removeEventListener("click", handleClick);
