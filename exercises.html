1. 基础练习

1.1 练习题

请编写一个函数 add，接受两个参数并返回它们的和。

1.2 答案

function add(a, b) {
    return a + b;
}

// 示例
console.log(add(5, 3)); // 输出: 8

2. 数组练习

2.1 练习题

请编写一个函数 getMax，接受一个数组并返回其中的最大值。

2.2 答案

function getMax(arr) {
    return Math.max(...arr);
}

// 示例
console.log(getMax([1, 5, 3, 9, 2])); // 输出: 9

3. 对象练习

3.1 练习题

请编写一个函数 getFullName，接受一个对象，包含 firstName 和 lastName 属性，并返回全名。

3.2 答案

function getFullName(person) {
    return `${person.firstName} ${person.lastName}`;
}

// 示例
const person = { firstName: "John", lastName: "Doe" };
console.log(getFullName(person)); // 输出: John Doe

4. Promise 练习

4.1 练习题

请创建一个返回延迟结果的 Promise 函数 delayedMessage，它在 2 秒后返回 "Hello, World!"。

4.2 答案

function delayedMessage() {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve("Hello, World!");
        }, 2000);
    });
}

// 示例
delayedMessage().then(message => console.log(message)); // 2秒后输出: Hello, World!
