1. 什么是 Promise?

Promise 是 JavaScript 中用于处理异步操作的一种对象，它表示一个可能尚未完成但最终会完成的操作。

2. Promise 的状态

Promise 有三种状态：

  • Pending（待定）：初始状态，既不是成功，也不是失败。
  • Fulfilled（已兑现）：操作成功完成。
  • Rejected（已拒绝）：操作失败。

3. 创建 Promise

使用 new Promise 构造函数创建 Promise 对象：

const myPromise = new Promise((resolve, reject) => {
    const success = true; // 模拟操作结果

    if (success) {
        resolve("操作成功！");
    } else {
        reject("操作失败！");
    }
});

4. 使用 Promise

使用 .then() 和 .catch() 方法处理 Promise 的结果：

myPromise
    .then(result => {
        console.log(result); // 处理成功结果
    })
    .catch(error => {
        console.error(error); // 处理失败结果
    });

5. Promise 的链式调用

可以将多个 Promise 进行链式调用：

const promise1 = new Promise((resolve) => {
    setTimeout(() => resolve("第一个 Promise 完成"), 1000);
});

const promise2 = new Promise((resolve) => {
    setTimeout(() => resolve("第二个 Promise 完成"), 500);
});

promise1
    .then(result1 => {
        console.log(result1);
        return promise2;
    })
    .then(result2 => {
        console.log(result2);
    });

6. Promise.all() 方法

使用 Promise.all() 方法并行执行多个 Promise，并在所有 Promise 完成后处理结果：

Promise.all([promise1, promise2])
    .then(results => {
        console.log("所有 Promise 完成:", results);
    })
    .catch(error => {
        console.error("其中一个 Promise 失败:", error);
    });
